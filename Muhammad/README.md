# Fazah Manual Human-Audit Suites — by Muhammad

Three courses, **same format**. Each folder is a complete audit suite you run inside Fazah.

| Course | Folder | Slides | Scenarios |
|---|---|---|---|
| Introduction to Software Engineering | [`software-engineering/`](./software-engineering/) | 5 decks (Ch1–Ch5) | 40 (6–16 turns) |
| Introduction to Database (Oracle PL/SQL) | [`introduction-to-database/`](./introduction-to-database/) | 8 lectures | 40 (6–16 turns) |
| Web Application Development (PHP) | [`web-application-development/`](./web-application-development/) | 15 decks | 40 (**10–30 turns** — deliberately longer/harder) |

**To run an audit:** open the course folder's `Fazah-Audit-Workbook.xlsx`, read its `README.md`, then
work the scenarios. The original slides are in each course's `course-files/`.
