# Fazah Manual Human-Audit Suites — by Muhammad

Two courses, **same format**. Each folder is a complete audit suite you run inside Fazah.

| Course | Folder | Slides | Scenarios |
|---|---|---|---|
| Introduction to Software Engineering | [`software-engineering/`](./software-engineering/) | 5 decks (Ch1–Ch5) | 40 multi-turn |
| Introduction to Database (Oracle PL/SQL) | [`introduction-to-database/`](./introduction-to-database/) | 8 lectures (Lec1–Lec7 + Lec5_1) | 40 multi-turn |

**To run an audit:** open the course folder's `Fazah-Audit-Workbook.xlsx`, read its `README.md`, then
work the scenarios. The original slides are in each course's `course-files/`.
