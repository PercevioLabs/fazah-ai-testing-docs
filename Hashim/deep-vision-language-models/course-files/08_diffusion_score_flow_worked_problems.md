## Topic Scope & Key Formulas

**DDPM forward (closed form)**
- α_t = 1 − β_t,    ᾱ_t = ∏_{i=1}^{t} α_i
- x_t = √ᾱ_t · x_0 + √(1−ᾱ_t) · ε,    ε ~ N(0, I)
- q(x_t | x_0) = N( √ᾱ_t · x_0,  (1−ᾱ_t) I )

**True posterior q(x_{t−1} | x_t, x_0) = N(μ̃_t, β̃_t I)**
- μ̃_t = ( √ᾱ_{t−1} (1−α_t) / (1−ᾱ_t) ) · x_0  +  ( √α_t (1−ᾱ_{t−1}) / (1−ᾱ_t) ) · x_t
- β̃_t = ( (1−ᾱ_{t−1}) / (1−ᾱ_t) ) · β_t

**DDPM sampling step**
- x_{t−1} = (1/√α_t) ( x_t − ((1−α_t)/√(1−ᾱ_t)) · ε_θ(x_t, t) ) + σ_t · z,    z ~ N(0,I),   σ_t² = β̃_t

**DDIM sampling step (η blends DDIM↔DDPM; η=0 deterministic)**
- x̂_0 = (1/√ᾱ_t) ( x_t − √(1−ᾱ_t) · ε_θ )
- σ_t = η · √((1−ᾱ_{t′})/(1−ᾱ_t)) · √(1 − ᾱ_t/ᾱ_{t′})
- x_{t′} = √ᾱ_{t′} · x̂_0 + √(1−ᾱ_{t′} − σ_t²) · ε_θ + σ_t · z

**Score ↔ noise relation**
- Gaussian N(μ, v):    s(x) = −(x − μ) / v
- Forward marginal:    ∇_{x_t} log q(x_t | x_0) = −ε / √(1−ᾱ_t)
- DSM target:    ∇_{x̃} log p_σ(x̃ | x) = (x − x̃) / σ²
- NCSN parameterization:    s_θ(x̃, σ) ≈ −ε / σ²

**Simple ε-loss (DDPM)**
- L_simple = E_{t, x_0, ε} [ ‖ ε − ε_θ(x_t, t) ‖² ]

**Flow Matching (linear interpolant, noise→data)**
- x_t = t · x_1 + (1−t) · x_0,    t ∈ [0,1]
- v_target = x_1 − x_0
- L_CFM = E [ ‖ v_θ(x_t, t) − (x_1 − x_0) ‖² ]
- Euler sampling: x_{t+Δt} = x_t + Δt · v_θ(x_t, t)

**Classifier-Free Guidance (CFG)**
- ε̂ = ε_θ(x_t, t, ∅) + s · ( ε_θ(x_t, t, c) − ε_θ(x_t, t, ∅) )

**Noise schedules**
- Linear: β_t = β_1 + ((t−1)/(T−1)) · (β_T − β_1)
- Cosine: ᾱ_t = f(t)/f(0),  f(t) = cos²( ((t/T + s)/(1+s)) · π/2 ),  s = 0.008
- SNR = ᾱ_t / (1 − ᾱ_t)

## Problem 1: DDPM forward — compute ᾱ_t and marginal variance

**Given:** A DDPM with T = 4 steps and linear schedule β = (β_1, β_2, β_3, β_4) = (0.1, 0.2, 0.3, 0.4).  Clean data point x_0 = 2.

**Find:**  (a) ᾱ_t for t = 1, 2, 3, 4.  (b) The mean and variance of q(x_3 | x_0).  (c) The SNR ᾱ_t / (1−ᾱ_t) at t = 3.

**Solution:**

(a)  α_t = 1 − β_t  ⇒  α = (0.9, 0.8, 0.7, 0.6).

- ᾱ_1 = 0.9
- ᾱ_2 = 0.9 · 0.8 = 0.72
- ᾱ_3 = 0.72 · 0.7 = 0.504
- ᾱ_4 = 0.504 · 0.6 = 0.3024

(b)  q(x_3 | x_0) = N( √ᾱ_3 · x_0,  (1−ᾱ_3) I )

- Mean = √0.504 · 2 = 0.70993 · 2 = **1.4199**
- Variance = 1 − 0.504 = **0.496**

(c)  SNR_3 = ᾱ_3 / (1 − ᾱ_3) = 0.504 / 0.496 = **1.0161**

**Answer:**  ᾱ = (0.9, 0.72, 0.504, 0.3024);  q(x_3|x_0) = N(1.4199, 0.496);  ⌐SNR_3 ≈ 1.016¬

## Problem 2: One-shot forward jump with a fixed ε

**Given:** Same schedule as Problem 1 (β = 0.1, 0.2, 0.3, 0.4).  Clean image x_0 = 1.5.  A specific noise sample ε = −0.8 is drawn (treat as scalar).

**Find:**  (a) x_2 generated by the one-shot formula x_t = √ᾱ_t x_0 + √(1−ᾱ_t) ε.  (b) The equivalent two-step formulation x_2 = √α_2 · x_1 + √β_2 · ε_2' produces the same marginal distribution — verify by computing the variance both ways given x_0.

**Solution:**

(a)  ᾱ_2 = 0.72 from Problem 1.

- √ᾱ_2 = √0.72 = 0.84853
- √(1−ᾱ_2) = √0.28 = 0.52915
- x_2 = 0.84853 · 1.5 + 0.52915 · (−0.8)
- x_2 = 1.27279 − 0.42332 = **0.84947**

(b)  Variance check given x_0 fixed:

- Marginal: Var(x_2 | x_0) = 1 − ᾱ_2 = 0.28.
- Two-step: x_1 = √α_1 x_0 + √β_1 ε_1, Var(x_1|x_0) = β_1 = 0.1.
  Then x_2 = √α_2 x_1 + √β_2 ε_2, so Var(x_2|x_0) = α_2 · Var(x_1|x_0) + β_2 = 0.8 · 0.1 + 0.2 = 0.08 + 0.2 = **0.28**.  ✓

**Answer:**  ⌐x_2 = 0.8495¬;  variance matches at 0.28 from both routes.

## Problem 3: Posterior mean μ̃_t and variance β̃_t

**Given:**  DDPM with β = (0.1, 0.2, 0.3) over T = 3.  At inference time you have the (training-side) pair x_0 = 1.0 and x_3 = 0.5.

**Find:**  (a) ᾱ_2 and ᾱ_3.  (b) The posterior variance β̃_3 = Var( x_2 | x_3, x_0 ).  (c) The posterior mean μ̃_3 = E[ x_2 | x_3, x_0 ].

**Solution:**

(a)  α = (0.9, 0.8, 0.7).  ᾱ_1 = 0.9,  ᾱ_2 = 0.72,  ᾱ_3 = 0.504.

(b)  β̃_3 = ((1 − ᾱ_2) / (1 − ᾱ_3)) · β_3
      = (0.28 / 0.496) · 0.3
      = 0.56452 · 0.3
      = **0.16935**

(c)  μ̃_3 = ( √ᾱ_2 · (1 − α_3) / (1 − ᾱ_3) ) · x_0  +  ( √α_3 · (1 − ᾱ_2) / (1 − ᾱ_3) ) · x_3

- Coefficient of x_0:  √0.72 · 0.3 / 0.496 = 0.84853 · 0.60484 = 0.51322
- Coefficient of x_3:  √0.7 · 0.28 / 0.496 = 0.83666 · 0.56452 = 0.47232
- μ̃_3 = 0.51322 · 1.0  +  0.47232 · 0.5
- μ̃_3 = 0.51322 + 0.23616 = **0.74938**

Sanity check: μ̃_3 lies between x_3 = 0.5 and x_0 = 1.0, pulled towards x_0 by the clean-image term — as expected.

**Answer:**  ⌐β̃_3 ≈ 0.1694,  μ̃_3 ≈ 0.7494¬

## Problem 4: DDPM reverse step (single step with noise)

**Given:**  At time t, you have α_t = 0.8, ᾱ_t = 0.4, ᾱ_{t−1} = 0.5, x_t = 1.0.  The network outputs ε_θ(x_t, t) = 0.3.  A standard normal sample z = 1.2 is drawn for the stochastic injection.  Use σ_t² = β̃_t.

**Find:**  (a) β_t.  (b) σ_t.  (c) The deterministic mean μ_θ.  (d) The full stochastic sample x_{t−1}.

**Solution:**

(a)  β_t = 1 − α_t = 1 − 0.8 = **0.2**

(b)  σ_t² = β̃_t = ((1 − ᾱ_{t−1}) / (1 − ᾱ_t)) · β_t
       = (0.5 / 0.6) · 0.2
       = 0.83333 · 0.2 = 0.16667
- σ_t = √0.16667 = **0.40825**

(c)  μ_θ = (1/√α_t) ( x_t − ((1 − α_t)/√(1 − ᾱ_t)) · ε_θ )

- 1/√α_t = 1/√0.8 = 1/0.89443 = 1.11803
- (1 − α_t)/√(1 − ᾱ_t) = 0.2/√0.6 = 0.2 / 0.77460 = 0.25820
- inside: 1.0 − 0.25820 · 0.3 = 1.0 − 0.07746 = 0.92254
- μ_θ = 1.11803 · 0.92254 = **1.03145**

(d)  x_{t−1} = μ_θ + σ_t · z = 1.03145 + 0.40825 · 1.2 = 1.03145 + 0.48990 = **1.52134**

**Answer:**  ⌐x_{t−1} ≈ 1.5213¬  (deterministic mean μ_θ ≈ 1.0315, σ_t ≈ 0.4082)

## Problem 5: Deterministic DDIM step (η = 0)

**Given:**  Trained DDPM with cumulative products ᾱ at three timesteps:  ᾱ_4 = 0.30,  ᾱ_2 = 0.72,  ᾱ_0 := 1.  You run a DDIM sampler that takes the trajectory  t = 4 → t′ = 2 → t″ = 0  with η = 0 (deterministic).  At t = 4 you have x_4 = 0.9 and ε_θ(x_4, 4) = 0.6.  Later you obtain ε_θ(x_2, 2) = −0.2 at the next stage.

**Find:**  (a) The clean-image estimate x̂_0 at t = 4.  (b) x_2 produced by the first DDIM jump.  (c) x_0 produced by the second DDIM jump (note t″ = 0 ⇒ ᾱ = 1 ⇒ x_0 = x̂_0(x_2, 2)).

**Solution:**

(a)  x̂_0(x_4, 4) = (1/√ᾱ_4) ( x_4 − √(1 − ᾱ_4) · ε_θ )

- √ᾱ_4 = √0.30 = 0.54772
- √(1 − ᾱ_4) = √0.70 = 0.83666
- x̂_0 = (1/0.54772) · ( 0.9 − 0.83666 · 0.6 )
- = 1.82574 · ( 0.9 − 0.50200 )
- = 1.82574 · 0.39800 = **0.72665**

(b)  With η = 0: x_{t′} = √ᾱ_{t′} · x̂_0 + √(1 − ᾱ_{t′}) · ε_θ

- √ᾱ_2 = √0.72 = 0.84853
- √(1 − ᾱ_2) = √0.28 = 0.52915
- x_2 = 0.84853 · 0.72665 + 0.52915 · 0.6
- = 0.61658 + 0.31749 = **0.93407**

(c)  At t″ = 0, ᾱ = 1, so x_0 equals the new x̂_0 computed from (x_2, 2):

- x̂_0(x_2, 2) = (1/√0.72) · ( 0.93407 − √0.28 · (−0.2) )
- = 1.17851 · ( 0.93407 + 0.52915 · 0.2 )
- = 1.17851 · ( 0.93407 + 0.10583 )
- = 1.17851 · 1.03990 = **1.22555**

**Answer:**  ⌐x̂_0 (at t=4) ≈ 0.7267,  x_2 ≈ 0.9341,  final x_0 ≈ 1.2256¬

## Problem 6: Score for a Gaussian + Denoising Score Matching loss

**Given:**  Clean data point x = 2.0.  Noise perturbation kernel p_σ(x̃ | x) = N(x, σ² I) with σ = 0.5.  A perturbed sample x̃ = 2.4 is drawn.  Your network outputs the score estimate ŝ_θ(x̃, σ) = −1.20 at this point.

**Find:**  (a) The DSM target ∇_{x̃} log p_σ(x̃ | x).  (b) The per-sample DSM loss (1/2) (ŝ − target)².  (c) Verify consistency with the NCSN parameterization s = −ε/σ², where ε = x̃ − x.  (d) If you had instead been told the marginal is exactly p(x) = N(0, v = 4) (no perturbation), what is the score s(x̃) at the same point?

**Solution:**

(a)  ∇_{x̃} log p_σ(x̃ | x) = (x − x̃) / σ²
- = (2.0 − 2.4) / (0.5)²
- = −0.4 / 0.25 = **−1.60**

(b)  Per-sample DSM loss:
- (1/2) (ŝ − target)² = (1/2) ( −1.20 − (−1.60) )²
- = (1/2) (0.40)² = 0.5 · 0.16 = **0.08**

(c)  ε = x̃ − x = 2.4 − 2.0 = 0.4.
- NCSN form: s = −ε / σ² = −0.4 / 0.25 = −1.60.  ✓  Identical to (a).

(d)  For p(x) = N(0, v = 4): s(x̃) = −x̃ / v = −2.4 / 4 = **−0.60**.

**Answer:**  ⌐target = −1.60,  DSM loss = 0.08,  Gaussian-marginal score = −0.60¬

## Problem 7: Flow Matching velocity and CFM loss

**Given:**  Noise sample x_0 = −1.0 (from p_0 = N(0,I)).  Data sample x_1 = 3.0.  Linear interpolant x_t = t · x_1 + (1−t) · x_0 with t ∈ [0,1].  Your flow network predicts velocities  v_θ(x_t, t)  shown below.

**Find:**  (a) x_t at t = 0.25 and at t = 0.75.  (b) The target velocity v(x_t | x_0, x_1).  (c) Per-sample CFM loss at t = 0.25 if v_θ = 3.5.  (d) One Euler sampling step starting from x_0 with step size Δt = 0.25 using the same v_θ = 3.5 — does it lie on the true interpolant at t = 0.25?

**Solution:**

(a)  x_t = t · x_1 + (1 − t) · x_0
- t = 0.25:  x_t = 0.25 · 3.0 + 0.75 · (−1.0) = 0.75 − 0.75 = **0.00**
- t = 0.75:  x_t = 0.75 · 3.0 + 0.25 · (−1.0) = 2.25 − 0.25 = **2.00**

(b)  Target velocity (conditional on the (x_0, x_1) pair):
- v = x_1 − x_0 = 3.0 − (−1.0) = **4.0** (constant in t for a straight-line interpolant)

(c)  L_CFM (single sample) = ( v_θ − v )² = ( 3.5 − 4.0 )² = (−0.5)² = **0.25**.

(d)  Euler step: x_{0+Δt} = x_0 + Δt · v_θ = −1.0 + 0.25 · 3.5 = −1.0 + 0.875 = **−0.125**.
- True interpolant at t = 0.25 is 0.00 (from part a).
- Discrepancy = 0.00 − (−0.125) = 0.125, caused by v_θ undershooting the true velocity of 4 by 0.5.

**Answer:**  ⌐v_target = 4.0,  CFM loss = 0.25,  Euler-step x_{0.25} = −0.125 (true value 0.00)¬

## Problem 8: Classifier-Free Guidance + DDPM step (mixed harder problem)

**Given:**  A trained conditional DDPM. At timestep t you have x_t = 1.5, α_t = 0.7, ᾱ_t = 0.3, ᾱ_{t−1} = 0.45.  The network outputs

- unconditional noise: ε_θ(x_t, t, ∅) = 0.10
- conditional noise:   ε_θ(x_t, t, c)  = 0.30

You run CFG with guidance scale s, then perform a DDPM reverse step (using the CFG-mixed noise) with σ_t = √β̃_t and z = 0.

**Find:**  (a) Compute β̃_t.  (b) Compute the CFG-mixed noise ε̂ for s ∈ {0, 1, 3}.  (c) Compute x_{t−1} for the same three guidance scales.  (d) Interpret in one sentence why x_{t−1} grows as s increases.

**Solution:**

(a)  β_t = 1 − α_t = 0.3.
- β̃_t = ((1 − ᾱ_{t−1}) / (1 − ᾱ_t)) · β_t  =  (0.55 / 0.70) · 0.3  =  0.78571 · 0.3 = **0.23571**.   (σ_t = √0.23571 ≈ 0.4855, unused because z = 0.)

(b)  ε̂(s) = ε_θ(·, ∅) + s · ( ε_θ(·, c) − ε_θ(·, ∅) ) = 0.10 + s · 0.20.

- s = 0:  ε̂ = **0.10** (purely unconditional)
- s = 1:  ε̂ = 0.10 + 0.20 = **0.30** (purely conditional)
- s = 3:  ε̂ = 0.10 + 0.60 = **0.70** (extrapolated)

(c)  μ_θ(s) = (1/√α_t) ( x_t − ((1 − α_t)/√(1 − ᾱ_t)) · ε̂ )

Precompute constants:
- 1/√α_t = 1/√0.7 = 1/0.83666 = 1.19523
- (1 − α_t)/√(1 − ᾱ_t) = 0.3 / √0.7 = 0.3 / 0.83666 = 0.35857

Now:
- s = 0:  inside = 1.5 − 0.35857 · 0.10 = 1.5 − 0.03586 = 1.46414
  μ_θ = 1.19523 · 1.46414 = **1.74999**
- s = 1:  inside = 1.5 − 0.35857 · 0.30 = 1.5 − 0.10757 = 1.39243
  μ_θ = 1.19523 · 1.39243 = **1.66428**
- s = 3:  inside = 1.5 − 0.35857 · 0.70 = 1.5 − 0.25100 = 1.24900
  μ_θ = 1.19523 · 1.24900 = **1.49285**

Since z = 0, x_{t−1} = μ_θ for each case.

(d)  Wait — observed pattern is x_{t−1} decreases as s increases (1.750 → 1.664 → 1.493). Interpretation: a larger guidance scale s amplifies the predicted noise ε̂, so the denoiser subtracts more from x_t at this step, pushing x_{t−1} further toward the conditional mode and away from x_t. This is exactly the “sharpening / extrapolation toward conditional” behaviour of CFG.

**Answer:**  ⌐β̃_t ≈ 0.2357;  (ε̂, x_{t−1}) at s = (0,1,3) is ( (0.10, 1.7500), (0.30, 1.6643), (0.70, 1.4928) )¬

## Quick-reference checks before exam
- ᾱ_t is monotonically decreasing in t (every α_i ∈ (0,1)).
- β̃_t < β_t always (since (1−ᾱ_{t−1}) < (1−ᾱ_t)).
- DDIM reduces to DDPM when η = 1 (σ_t = √β̃_t) and t′ = t−1.
- Score and noise: ∇_{x_t} log q(x_t|x_0) = −ε / √(1−ᾱ_t).
- Flow-matching with linear interpolant: v_target is constant in t along a given (x_0, x_1) pair.
- CFG: s = 0 ⇒ unconditional, s = 1 ⇒ conditional, s > 1 ⇒ extrapolation/sharpening.
