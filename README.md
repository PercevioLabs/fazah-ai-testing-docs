# Fazah AI — Testing Docs

Manual human-audit suites for the **Fazah** teacher chat agent — one per course. Each suite is 40
realistic multi-turn scenarios a human auditor runs inside Fazah, plus the original course slides.

## ➡ Everything is in [`Muhammad/`](./Muhammad/)

| Course | Folder | Slides | Scenarios |
|---|---|---|---|
| Introduction to Software Engineering | [`Muhammad/software-engineering/`](./Muhammad/software-engineering/) | 5 decks (Ch1–Ch5, Sommerville) | 40 multi-turn |
| Introduction to Database (Oracle PL/SQL) | [`Muhammad/introduction-to-database/`](./Muhammad/introduction-to-database/) | 8 lectures (Lec1–Lec7 + Lec5_1) | 40 multi-turn |

Each course folder has its own `README.md`, an Excel workbook (`Fazah-Audit-Workbook.xlsx`) with
dropdowns and every scenario/turn pre-filled, the 40 scenario files, a `misc/` reference folder, and
`course-files/` (the original slides).
